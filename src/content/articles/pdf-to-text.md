## Why You Need a PDF to Text

## How PDF to Text Works

The engine extracts text character vectors from PDF page streams using client-side parsers for high-speed operation, providing instant feedback without sending any data over the internet.

## When to Use This Tool

### Can it extract text from scanned images?

This tool extracts selectable digital text. For scanned paper documents, OCR (Optical Character Recognition) is required.
